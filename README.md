# 🔥 21 Days Hard Challenge Tracker

A simple and interactive web application designed to help users build consistency by tracking daily habits over a 21-day challenge.

---

## 🚀 Features

* ✅ Task management (add, edit, complete tasks)
* 📅 Date-based progression system
* 🔒 Future date locking before challenge starts
* 📊 Progress bar and streak tracking
* 💾 Local storage for persistent user data
* 🎯 Clean, minimal, and responsive UI

---

## 🛠️ Tech Stack

* **HTML**
* **CSS**
* **JavaScript**

---

## 💡 Description

The **21 Days Hard Challenge Tracker** helps users stay disciplined by completing daily tasks over a structured 21-day period.

* Each day unlocks sequentially based on the selected start date
* Users can manage tasks for each day
* Progress is tracked visually with a streak counter and progress bar
* All data is stored locally using browser storage (no backend required)
